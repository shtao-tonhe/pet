核心业务目标
实现商品购买全流程闭环，支持优惠券、积分抵扣，保障交易安全。
业务流程
购物车业务补充：
商品加入：商品详情页点击「加入购物车」→ 选择规格（如食品口味、项圈尺寸）→ 输入数量 → 加入成功（弹窗提示）
库存预警：商品库存低于 5 件时，购物车显示「库存紧张」标签
价格同步：购物车商品价格实时同步商城最新价格（含活动价）
订单创建业务：
结算流程：购物车选中商品 → 点击「结算」→ 选择收货地址 → 选择优惠券 / 积分抵扣（如有）→ 确认订单金额 → 选择支付方式（微信 / 支付宝）→ 提交订单
订单生成：系统生成唯一订单号，锁定商品库存（30 分钟内未支付自动释放）
支付业务：
对接第三方支付服务 → 跳转支付页面 → 支付成功 / 失败回调 → 订单状态更新（待发货 / 支付失败）
支付超时：30 分钟未支付自动取消订单，发送「订单已取消」消息通知
订单售后业务：
退款流程：申请退款 → 上传凭证（如商品质量问题照片）→ 商家审核 → 审核通过 → 退款至原支付账户
换货流程：申请换货 → 填写换货原因 → 商家审核 → 审核通过 → 用户寄回商品 → 商家确认收货 → 寄出新品
