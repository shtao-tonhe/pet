
## 开发资源

### 用户表（user）
| 字段 | 类型 | 说明 |
|------|------|------|
| id | string (主键) | 如 `u_1234567890` |
| phone | string | 唯一索引 |
| password_hash | string | BCrypt 加密后的密码 |
| is_deleted | boolean | 默认 false |
| created_at | timestamp |  |
| updated_at | timestamp |  |

### 短信验证码表（sms_code）
| 字段 | 类型 | 说明 |
|------|------|------|
| id | string |  |
| phone | string |  |
| code | string | 6位数字 |
| purpose | string | REGISTER / LOGIN / RESET_PASSWORD |
| expires_at | timestamp | 创建时间 + 5分钟 |
| used | boolean | 是否已被使用（防止重复使用） |
| created_at | timestamp |  |

> 建议：每次验证成功后将 `used = true`，防止重放攻击
